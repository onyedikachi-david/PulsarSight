# PulsarSight - SOON Network Explorer 🔭

PulsarSight is a powerful blockchain explorer designed specifically for the SOON Network, providing users with a comprehensive and intuitive interface to explore and analyze blockchain data.

## Features

- 📊 Real-time blockchain data visualization
- 🔍 Transaction tracking and analysis
- 📦 Block explorer with detailed information
- 👤 Account/Address lookup and monitoring
- 📈 Network statistics and metrics
- 🔗 Smart contract interaction and verification

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Web browser (Chrome, Firefox, Safari, or Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/onyedikachi-david/PulsarSight.git
cd PulsarSight
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

The application will be available at `http://localhost:3000`
